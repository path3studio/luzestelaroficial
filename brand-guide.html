Seccion 4

Simbolos del Zodiaco

Los 12 signos se representan con glifos Unicode renderizados como texto dorado (nunca como emoji). Se aplica el selector de variacion de texto (︎) y la fuente Noto Sans Symbols 2 para asegurar una representacion consistente.

/* Renderizado de simbolos zodiacales */
font-family: 'Noto Sans Symbols 2', serif;
font-variant-emoji: text;
background: linear-gradient(135deg, #f0d078, #d4a849, #c89030);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

Seccion 5

Glass Morphism

Todos los componentes de tarjeta usan un estilo de vidrio esmerilado que refuerza la estetica cosmica. Hay tres niveles de elevacion según la importancia del contenido.

Estandar

Tarjetas de contenido general, zodiac cards, feature cards. El nivel base de vidrio esmerilado.

background: rgba(255,255,255,0.04);
border: 1px solid rgba(255,255,255,0.08);
border-radius: 16px - 18px;
backdrop-filter: blur(12px);

Elevado

Tarjetas destacadas, modales, tooltips. Mayor opacidad y blur para crear profundidad visual.

background: rgba(255,255,255,0.06);
border: 1px solid rgba(255,255,255,0.12);
border-radius: 18px - 24px;
backdrop-filter: blur(16px);

Acento

Secciones especiales como "Proximamente" o contenido premium. Combina purpura y oro en el gradiente.

background: linear-gradient(135deg, rgba(124,92,191,0.1), rgba(212,168,73,0.08));
border: 1px solid rgba(124,92,191,0.2);
border-radius: 24px;
backdrop-filter: blur(12px);

Variables CSS Compartidas

--glass: rgba(255,255,255,0.04);
--glass-border: rgba(255,255,255,0.08);

/* Hover en tarjetas */
transition: transform 0.3s, border-color 0.3s;
/* :hover */
transform: translateY(-3px);
border-color: rgba(212,168,73,0.25);

Seccion 6

Reglas de Uso

Directrices fundamentales para mantener la coherencia visual de Luz Estelar en todas las plataformas y materiales.

Correcto

Fondos oscuros siempre

Usar exclusivamente fondos oscuros (#06061a, #0c0c2a o tonos similares). El contenido de Luz Estelar vive en el espacio profundo.

Incorrecto

Fondos claros o blancos

Nunca usar fondos blancos, crema o grises claros. Esto rompe completamente la identidad cosmica de la marca.

Correcto

Oro como acento primario

El dorado es el color de acento dominante. Usarlo en titulos, iconos, bordes hover y elementos de enfasis.

Incorrecto

Otros colores calidos como primario

Nunca usar rojo, naranja o amarillo saturado como color de acento principal. Estos se reservan solo para los elementos zodiacales (Fuego).

Correcto

Simbolos como texto

Renderizar los simbolos zodiacales como glifos de texto con la fuente Noto Sans Symbols 2 y el selector de variacion ︎.

Incorrecto

Simbolos como emoji

No permitir que los simbolos zodiacales se rendericen como emojis de color. Esto pierde la estetica dorada y elegante de la marca.

Correcto

Tamanos minimos de fuente

Texto de cuerpo: minimo 14px. Etiquetas y labels: minimo 12px. Esto garantiza legibilidad sobre fondos oscuros.

Correcto

Imagenes cosmicas y celestiales

Toda imagen, ilustracion o fondo debe evocar el cosmos: estrellas, nebulosas, constelaciones, gradientes espaciales. La marca vive en el universo.
